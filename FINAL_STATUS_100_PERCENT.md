## ✅ DEPLOYMENT STEPS

### 1. Install Dependencies
```bash
npm install
```

### 2. Run Tests
```bash
npm test                    # All tests
npm test -- --coverage     # With coverage
```

### 3. Create Database Indexes
```bash
# Automatically created by Mongoose on connection
# Or manually:
db.coupons.createIndex({ code: 1, isActive: 1 })
db.coupons.createIndex({ expiryDate: 1, isActive: 1 })
db.couponusages.createIndex({ couponCode: 1, createdAt: -1 })
db.couponusages.createIndex({ userId: 1, createdAt: -1 })
db.couponusages.createIndex({ orderId: 1 })
```

### 4. Start Application
```bash
npm start              # Production
npm start:dev         # Development
```

### 5. Verify Health
```bash
curl http://localhost:3000/coupons/available \
  -H "Authorization: Bearer <JWT>"
```
